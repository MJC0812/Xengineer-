# AI 语音绘图工具

一个纯语音控制的智能绘图工具，支持多种方言和自然语言指令，让您无需使用鼠标或键盘即可完成绘图创作。

## 功能特性

### 🎤 语音控制
- **纯语音交互**：通过语音指令完成所有绘图操作
- **多方言支持**：普通话、粤语、四川话、河南话、东北话、台湾话、上海话、英语
- **语音唤醒**：支持"小助手"唤醒词
- **智能纠错**：拼音相似度匹配，容忍口音和口语表达

### 🎨 绘图功能
- **基础工具**：画笔、橡皮擦
- **几何图形**：圆形、正方形、三角形、矩形、椭圆、菱形、平行四边形、心形、圆环、多边形
- **数学函数**：正弦函数(sin)、余弦函数(cos)、正切函数(tan)、抛物线、极坐标方程
- **专业绘图**：贝塞尔曲线、箭头、直线、带箭头的直线
- **AI 智能绘图**：自然语言描述绘图、简笔画生成

### 🎯 智能特性
- **意图分析**：智能识别日常对话和绘图指令
- **上下文理解**：支持复杂指令的拆解与执行
- **实时反馈**：语音合成朗读操作结果
- **多轮对话**：支持连续语音交互

### 🎭 界面功能
- **实时波形显示**：可视化音频输入
- **麦克风音量指示**：实时音量监控
- **对话记录**：显示语音识别结果和操作历史
- **主题切换**：支持浅色/深色主题
- **响应式布局**：适配不同屏幕尺寸

## 技术架构

### 前端技术
- **HTML5 Canvas**：高性能 2D 绘图
- **Web Speech API**：原生语音识别与合成
- **CSS3**：现代化界面设计，带毛玻璃效果和动画
- **离屏渲染**：优化绘图性能

### 核心模块
- `VoiceDrawingApp`：主应用类
- `IntentAnalyzer`：意图分析模块
- `PinyinMatcher`：拼音相似度匹配
- `MathExpressionParser`：数学表达式解析
- `DrawingEngine`：绘图引擎

## 使用方法

### 启动应用
1. 在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：`python -m http.server 8000`，然后访问 `http://localhost:8000`

### 基本操作
1. 点击麦克风按钮开始语音输入
2. 说出绘图指令，例如：
   - "画一个圆"
   - "画红色的正方形"
   - "画 sinx 函数"
   - "切换到橡皮擦"
   - "撤销"
3. 系统会自动识别并执行绘图操作

### 语音指令示例
| 功能 | 语音指令 |
|------|---------|
| 绘制图形 | 画圆、画三角形、画正方形、画心形 |
| 设置颜色 | 红色、蓝色、绿色、黄色 |
| 设置大小 | 粗细 5、笔刷大小 10 |
| 数学函数 | sinx、cosx、y=x²、极坐标 |
| 工具切换 | 橡皮擦、画笔、填充模式 |
| 历史操作 | 撤销、重做、清空画布 |

### 扩展开发
如需添加新功能，请遵循以下规范：
1. 在 `VoiceDrawingApp` 类中添加独立的绘图函数，使用 `draw` 前缀命名
2. 在方言配置中添加对应的语音关键词
3. 在 `commandMap` 中关联指令与绘制函数
4. 补充语音反馈文案
5. 使用 `saveState()` 记录操作以支持撤销/重做

## 注意事项

### 权限要求
- 需要麦克风权限才能使用语音识别功能
- 首次使用时，浏览器会弹出权限请求，请允许使用麦克风

### 使用环境
- 推荐在安静的环境下使用
- 麦克风距离嘴部约 15 厘米效果最佳
- 避免正对扬声器或风扇

### 性能优化
- 使用离屏 Canvas 渲染优化绘图性能
- 支持批量绘制操作
- 自适应窗口大小和缩放

## 技术支持

如遇问题，请检查：
1. 浏览器是否支持 Web Speech API
2. 麦克风权限是否已授权
3. 网络连接是否正常（部分功能需要网络）
